# Book to Course / 化书成课

## 中文

Book to Course 是一个把书籍、手稿、读书笔记、PDF、DOCX 或章节大纲转化为课程方案的 Codex skill。它适合把知识内容重构成可教学、可练习、可交付的课程产品，尤其适用于中文成人学习、企业培训、工作坊和知识付费课程。

### 适合场景

- 把一本书拆成系统课程
- 把读书笔记升级为培训课纲
- 设计模块、课时、练习、案例讨论和讲师备注
- 生成学员手册、工作表和课程交付材料
- 为后续视觉课件生成提供结构化内容

### 核心能力

- 提炼课程定位、学习对象和学习成果
- 设计模块结构、课时安排和教学路径
- 将书中概念转化为活动、练习、案例和输出任务
- 区分学员材料、讲师备注和课堂引导
- 保持课程可教、可练、可复用

### 使用方式

把本仓库作为一个 Codex skill 安装或复制到你的 Codex skills 目录，然后在 Codex 中说：

```text
Use $book-to-course 把这本书/这份笔记转成一门课程：……
```

## English

Book to Course is a Codex skill for transforming books, manuscripts, notes, PDFs, DOCX files, or outlines into teachable course products. It is designed for adult learning, workshops, corporate training, and structured knowledge products.

### Best For

- Turning a book into a structured course
- Converting reading notes into a training syllabus
- Designing modules, lessons, practices, case discussions, and instructor notes
- Creating learner handbooks, worksheets, and delivery materials
- Preparing structured content for downstream visual courseware

### What It Does

- Clarifies positioning, audience, and learner outcomes
- Designs modules, lesson flow, and teaching sequence
- Converts concepts into activities, practices, cases, and outputs
- Separates learner-facing materials from instructor guidance
- Keeps the course teachable, practical, and reusable

### Usage

Install or copy this repository into your Codex skills directory, then ask Codex:

```text
Use $book-to-course turn this book or reading note into a course: ...
```
